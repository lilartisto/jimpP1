2 argument to plik wejsciowy, jesli sie nie poda to program bedzie czytac z pliku "data.txt"
3 argument to do jakiego pliku zapisac koncowy stan planszy

Plik wejsciowy i plik koncowy planszy zapisywane sÄ… w takim samym formacie:

szerokosc wysokosc
x1 y1
x2 y2
...
xn yn

Punkt poczatkowy planszy to (1, 1), a koncowy to (szerokosc, wysokosc)

Obrazy sa numerowane od 0 do n( n - ilosc iteracji), gdzie "photo0.png" to stan poczatkowy po wczytaniu z pliku
